	\subsection{Suffix Automaton}
		\begin{flushleft}
			需要注意的知识点：
			\begin{itemize}
				\item 每个状态对应一个右端点right集合（等价类），串的长度为[min, max]，短了集合变大，长了集合变小
				\item parent树，根为全集，越往下串长越长，max(父亲) + 1 == min(儿子)
				\item 子串是后缀的前缀，SAM经常被用来判断子串/统计子串个数
			\end{itemize}
		\end{flushleft}
		\lstinputlisting{Sections/String/SAM.cpp}
